# Pusakat Management API

A professional,  REST API for cat record management.

---

## Quick Setup

1. **Initialize:** Run `php spark migrate`
2. **Start Server:** Run `php spark serve`

## Verification & Results

All testing scenarios, security roles, and API "Answers" are documented as **Screenshots and Videos** in the **[Postman](./PostMan)** folder.

---

## Security Summary

- **Admin (`admin:admin`):** Full CRUD permissions.
- **Staff (`staff:staff`):** Read-only permissions.
- **Protocol:** Basic Authentication (JSON).
